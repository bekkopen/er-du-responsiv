Responsive Web Design

Fleksibel og enhets-uavhengig design

Ikke mobildesign eller design for spesifikke enheter!

  • Umulig å ha oversikt over alle mulige skjermstørrelser på mobiltelefoner, lesebrett, netbooks, laptoper, stasjonære…
  • Kan du se inn i fremtiden? Hva slags enheter og skjermstørrelser vil komme?
  • Er det riktig at vi som utviklere begrenser hvilke nettlesere/skjermstørrelser besøkende kan bruke?
The primary design principle underlying the Web’s usefulness and growth is universality. […] And it should be accessible from any kind of hardware that can connect to the Internet: stationary or mobile, small screen or large.
Tim Berners–Lee


  1. Acknowledge and embrace unpredictability.
  2. Think and behave in a future-friendly way.
  3. Help others do the same.

CSS3 media queries

  #page > article,
  #page > aside {
    float: none;
    width: auto;
  }
  #page > article {
    margin-right: 0;
  }
}

Responsive bilder 

<noscript data-widths="400,600,800" data-base-src="responsive-.png">
  <img src="river-400.png">
</noscript>
Eksempel 
$(function(){
  $("noscript").each(function() {
    var ns = $(this),
        best_fit = 0,
        content_width = ns.parent().width(),
        img_widths = $.map(ns.data("widths").split(","),
                           function(width) { return parseInt(width) });

    $.each(img_widths, function(i, img_width) {
      best_fit = i;
      return img_width < content_width;
    });

    var src = ns.data("base-src").replace(/(.+)(\.\w{3})/,
                      "$1" + img_widths[best_fit] + "$2")
    ns.after($("<img>").attr("src", src));
  });
});
