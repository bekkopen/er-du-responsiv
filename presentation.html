Media queries

Kilde: http://flic.kr/p/62qUVi

CSS2 media typer 

<link rel="stylesheet" media="screen" href="/screen.css">
<link rel="stylesheet" media="print" href="/print.css">
@media screen {
  body {
    background: #000;
    color: #fff;
  }
}
@media print {
  body {
    background: #fff;
    color: #000;
  }
}

CSS3 media queries 

<link rel="stylesheet" media="screen and (max-width: 35em)"
      href="/narrow.css">
@media screen and (max-width: 35em) {
  #page > article,
  #page > aside {
    float: none;
    width: auto;
  }
  #page > article {
    margin-right: 0;
  }
}
